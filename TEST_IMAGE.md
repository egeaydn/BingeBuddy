# Image Test

Test this URL in your browser:
https://image.tmdb.org/t/p/w500/fBmLlHKvDuvO4316qzqaDxEuKsg.jpg

If you see an image, the TMDB API images are working.

## Debug Steps:
1. Open browser console (F12)
2. Look for "MovieCard Debug" logs
3. Check Network tab for failed image requests
4. Look for CORS errors or 403/404 status codes
